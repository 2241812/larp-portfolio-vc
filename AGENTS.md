# Portfolio Custom Agents

This project has specialized agents for different tasks. Use `@agent-name` to invoke them.

## Available Agents

| Agent | Description | How to Use |
|-------|-------------|------------|
| `@ui-ux` | UI/UX and frontend specialist | `@ui-ux Create a new animation for the hero section` |
| `@3d-visual` | 3D graphics and visual effects | `@3d-visual Add particle effects to the background` |
| `@content` | Content and resume updates | `@content Update my skills section with new technologies` |
| `@code-quality` | Code refactoring and best practices | `@code-quality Refactor the page component for better performance` |
| `@feature` | New features and pages | `@feature Add a blog section to the portfolio` |

## Primary Agent

Use Tab to switch between:
- **Build**: Full development with all tools
- **Plan**: Analysis and planning without making changes
